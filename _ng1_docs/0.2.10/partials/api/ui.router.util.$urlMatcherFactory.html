$urlMatcherFactory (service in module ui.router.util )

Description

Factory for ui.router.util.type:UrlMatcher instances. The factory is also available to providers under the name $urlMatcherFactoryProvider.

Methods

  • caseInsensitiveMatch(value)

    Define if url matching should be case sensistive, the default behavior, or not.

    Parameters

    • value – {bool} –

      false to match URL in a case sensitive manner; otherwise true;

  • compile(pattern)

    Creates a ui.router.util.type:UrlMatcher for the specified pattern.

    Parameters

    • pattern – {string} –

      The URL pattern.

    Returns

    {ui.router.util.type:UrlMatcher} –

    The UrlMatcher.

  • isMatcher(object)

    Returns true if the specified object is a UrlMatcher, or false otherwise.

    Parameters

    • object – {Object} –

      The object to perform the type check against.

    Returns

    {Boolean} –

    Returns true if the object has the following functions: exec, format, and concat.
